# Peg Solitare Board

A replication of a 7x7 peg solitare board for my CS 627 Artificial Intelligence Class:

```
    O O O    
    O O O
O O O O O O O
O O O . O O O
O O O O O O O
    O O O
    O O O
```

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd Peg-Solitare
```

Start the solitare in game mode

```bash
  py solitare.py game
```
